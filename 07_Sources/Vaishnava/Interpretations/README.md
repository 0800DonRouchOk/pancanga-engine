# Interpretations

Este directorio registra fuentes secundarias o prácticas sobre reglas
Vaiṣṇavas.

Las interpretaciones pueden orientar la especificación, pero no reemplazan
Hari-bhakti-vilāsa como autoridad normativa primaria.

## Fuente Secundaria Referida

```text
Ramanuja Das / masterhindu.com
```

Estado:

```text
URL verificable pendiente.
Nivel de evidencia: secundario / provisional.
```

Uso permitido:

- orientar preguntas normativas;
- identificar datos calendáricos necesarios;
- preparar casos de investigación.

Uso no permitido:

- cerrar reglas;
- desbloquear implementación;
- sustituir fuentes primarias.
