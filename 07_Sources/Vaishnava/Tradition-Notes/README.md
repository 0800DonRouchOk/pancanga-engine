# Tradition Notes

Este directorio registra notas interpretativas de la tradición seguida por
Pancanga Engine, especialmente cuando ayuden a interpretar pasajes de
Hari-bhakti-vilāsa.

## Alcance

```text
Nityānanda Vaṁśa
```

Las notas de tradición pueden aclarar cómo aplicar una regla, pero no reemplazan
la autoridad normativa primaria.

## Relación Con Fuentes

| Capa | Función |
|---|---|
| Hari-bhakti-vilāsa | Fuente normativa primaria |
| Digdarśinī-ṭīkā | Comentario tradicional para ambigüedades |
| Navadvīpa Pañjikā | Aplicación Gauḍīya práctica |
| Tradition-Notes | Interpretación de la línea seguida por el proyecto |

## Regla

```text
Una nota de tradición puede orientar la implementación solo cuando esté
vinculada a una regla HBV-EK y a un pasaje primario identificado.
```
