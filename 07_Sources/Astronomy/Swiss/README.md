# Swiss Ephemeris

Autoridad: oráculo astronómico de validación.

Swiss Ephemeris no es el motor de cálculo de Pancanga Engine.

Uso:

- comparar longitudes solares;
- comparar longitudes lunares;
- comparar eventos astronómicos cuando corresponda;
- detectar diferencias sistemáticas.

Regla:

```text
Si Pancanga Engine difiere de Swiss Ephemeris, primero se investiga la causa.
No se modifica el motor para igualar el oráculo sin explicación científica.
```
