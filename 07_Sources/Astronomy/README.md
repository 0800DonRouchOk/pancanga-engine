# Astronomy Sources

Este directorio registra las fuentes astronómicas usadas por Pancanga Engine.

## Autoridades

- Meeus: referencia algorítmica principal para el motor solar, tiempo,
  aberración y nutación truncada.
- ELP2000: fuente primaria para la teoría lunar.
- Swiss Ephemeris: referencia externa de validación, no motor de cálculo.

## Estado

```text
Meeus      Primaria / algorítmica
ELP2000    Primaria / teoría lunar
Swiss      Oráculo de validación
```
