# PES-0001: Project Foundation

Status: 🔒 Congelado

Version: 0.1.0 Alpha

## Summary

This specification establishes the initial structure for Pancanga Engine, an
offline Gaudiya calendar engine planned for Rust.

## Scope

This foundation includes repository organization, documentation placeholders,
research areas, data folders, testing areas, and a Rust workspace skeleton.

## Foundational Documents

- PES-0001: Project Foundation
- ARCH-0001: Arquitectura General

## Non-Goals

- No calendar rules are implemented.
- No astronomical algorithms are implemented.
- No public API is finalized.
- No reference data is bundled yet.
