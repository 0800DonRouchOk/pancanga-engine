# RFC

Las RFC son documentos de discusión para ideas abiertas que todavía no deben
convertirse en una decisión arquitectónica (ADR) ni en una especificación
normativa (PES).

## Uso

Crear una RFC cuando una pregunta técnica necesite exploración antes de
convertirse en decisión.

## Ejemplos

- ¿Debemos soportar múltiples ayanāṁśas?
- ¿Qué precisión mínima debe exigir el modelo lunar?
- ¿Qué fuentes externas serán referencias de regresión?
