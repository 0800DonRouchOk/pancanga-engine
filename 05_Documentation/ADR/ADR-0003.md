# ADR-0003: Modelo Astronómico Basado en Meeus y Ayanāṁśa Lahiri

Estado: 🟡 En revisión

Fecha: 11 de julio de 2026

## Contexto

Pancanga Engine necesita una base astronómica oficial para evitar que cada
módulo elija fórmulas, precisión y fuentes de referencia de manera aislada.

El subsistema astronómico deberá ser reproducible, documentado, offline y lo
suficientemente claro para ser validado contra referencias publicadas.

## Decisión en Revisión

El motor astronómico oficial de Pancanga Engine se basará en:

- Jean Meeus, *Astronomical Algorithms*, 2nd Edition.
- Ayanāṁśa Lahiri para la capa sidérea por defecto.

Meeus será la referencia primaria para los algoritmos astronómicos iniciales.
Lahiri será el modelo sidéreo predeterminado cuando el cálculo requiera
ayanāṁśa.

## Alcance

Esta decisión afectará a:

- Julian Centuries.
- Modelo solar.
- Nutación.
- Oblicuidad.
- Tiempo sidéreo.
- Modelo lunar.
- Ayanāṁśa.

## Consecuencias

- El roadmap astronómico podrá dividirse en algoritmos pequeños y verificables.
- Cada algoritmo deberá tener una ficha técnica propia.
- Las pruebas deberán incluir ejemplos publicados cuando existan.
- Las diferencias frente a SOFA, Swiss Ephemeris u otras referencias deberán
  documentarse cuando correspondan.

## Estado de Revisión

ADR-0003 no está aprobado todavía.

Antes de aprobarlo se deberá revisar:

- si Meeus es suficiente para la precisión mínima requerida;
- cómo se documentará la compatibilidad con PureBhakti;
- qué partes deberán compararse contra SOFA u otras referencias;
- qué definición exacta de Lahiri se adoptará como predeterminada.
