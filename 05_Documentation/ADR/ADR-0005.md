# ADR-0005: Modelo de Aberración Solar

Estado: Aceptado

Fecha: 11 de julio de 2026

## Contexto

Pancanga Engine ya adoptó `VSOP87D` como base para la cadena solar y ya calcula
dos magnitudes físicas necesarias:

- longitud geocéntrica del Sol;
- radio vector Tierra-Sol en unidades astronómicas.

El siguiente fenómeno físico del pipeline solar es la aberración solar. Para el
objetivo del proyecto, calcular Ekādaśī offline con una base auditable, no es
necesario introducir todavía toda la infraestructura de resoluciones IAU
2000/2006.

## Decisión

Pancanga Engine implementará la corrección de aberración solar siguiendo
*Astronomical Algorithms*, 2nd edition, de Jean Meeus.

La corrección adoptada será:

```text
Δλ = -20.4898" / R
```

donde `R` es el radio vector Tierra-Sol en unidades astronómicas.

## Justificación

- Es compatible con la cadena solar basada en `VSOP87D`.
- Está ampliamente documentada en la bibliografía usada por el proyecto.
- Es sencilla de auditar.
- Su precisión es suficiente para la etapa solar requerida por el cálculo de
  Ekādaśī.
- Mantiene una separación clara entre aberración y nutación.

## Alternativas Consideradas

### Infraestructura IAU 2000/2006

Descartada para esta etapa. Aunque es más completa, introduce complejidad
prematura para el alcance actual del motor.

### Fórmula monolítica de longitud aparente

Descartada. Pancanga Engine mantiene cada corrección física como un algoritmo
separado para facilitar auditoría, pruebas y reemplazos futuros.

## Consecuencias

- `AST-0005` queda desbloqueado.
- La aberración solar se modelará como una corrección angular explícita.
- La nutación en longitud seguirá siendo un algoritmo separado.
- La longitud aparente final del Sol se construirá recién cuando existan las
  correcciones necesarias.
