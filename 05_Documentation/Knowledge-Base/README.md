por Pancanga Engine.

Estos documentos no reemplazan las fuentes primarias. Su función es convertir
fuentes, decisiones y notas de investigación en conocimiento auditable para el
proyecto.

## Regla De Uso

Un documento de Knowledge Base puede orientar especificaciones e
implementaciones, pero solo desbloquea código cuando sus reglas citan una
fuente primaria identificable o una decisión explícita del proyecto.

## Freeze Rule

Desde [Knowledge Base v1.0 Release](Knowledge-Base-v1.0-Release.md), ninguna
modificación a la Knowledge Base puede hacerse para facilitar la implementación.
Solo puede modificarse si una fuente primaria obliga al cambio.
