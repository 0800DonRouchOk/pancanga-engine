# HBV Citation Normalization

Estado: 🟡 Activo

## Propósito

Durante la investigación inicial se utilizaron referencias provenientes de
fuentes secundarias. Algunas referencias circularon como si pertenecieran al
Vilāsa 12, por ejemplo:

```text
HBV 12.293-295
```

Con la incorporación de la traducción íntegra de Bhṛgumuni Dāsa y la numeración
de la edición Haridāsa Śāstrī, esas referencias deben normalizarse antes de
entrar en la Knowledge Base.

## Decisión

La numeración de referencia oficial del proyecto para reglas de Ekādaśī será:

```text
Hari-bhakti-vilāsa
Edición Haridāsa Śāstrī
Traducción íntegra de Bhṛgumuni Dāsa
```

## Regla

```text
Toda cita secundaria debe verificarse y normalizarse contra esta numeración
antes de congelar una regla HBV-EK.
```

## Regla Sobre Búsquedas Web

Las búsquedas web pueden localizar o verificar fuentes, pero no fundamentan
reglas normativas.

Una regla HBV-EK solo puede congelarse desde:

- Hari-bhakti-vilāsa;
- Digdarśinī-ṭīkā, cuando aclare el texto;
- fuentes de aplicación tradicional, solo como confirmación práctica.

## Consecuencia

Las citas antiguas no se eliminan automáticamente, pero no pueden usarse como
referencia normativa hasta quedar mapeadas a la numeración oficial del
proyecto.
