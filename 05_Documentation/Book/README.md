# Libro Técnico de Pancanga Engine

Este directorio contiene la estructura inicial del libro técnico del proyecto.

El libro documentará la historia, arquitectura, matemática, astronomía y reglas
calendáricas del motor a medida que las especificaciones sean revisadas y
aprobadas.

## Capítulos

1. Historia
2. Arquitectura
3. Tiempo
4. Julian Day
5. Sol
6. Luna
7. Zonas Horarias
8. Ayanamsa
9. Tithi
10. Naksatra
11. Yoga y Karana
12. Masa y Gaurabda
13. Festivales
14. Parana
15. Ekadashi
