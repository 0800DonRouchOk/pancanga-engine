# 1. Objetivo

Convertir la longitud heliocéntrica de la Tierra en la longitud geocéntrica del
Sol.

Este algoritmo invierte el punto de vista: desde la Tierra, el Sol se encuentra
en la dirección opuesta a la posición heliocéntrica de la Tierra.

---

# 2. Derivación Geométrica

Sea:

```text
L = longitud heliocéntrica de la Tierra
```

La longitud geocéntrica del Sol es la dirección opuesta sobre la eclíptica:

```text
Lg = L + 180°
```

Como las longitudes eclípticas son cíclicas, el resultado debe normalizarse:

```text
Lg = normalize_360(L + 180°)
```

---

# 3. Fórmula Implementada

```text
Lg = normalize_360(L + 180°)
```

No se aplica:

- aberración;
- nutación;
- corrección aparente;
- ayanāṁśa;
- reglas de calendario.

---

# 4. API Pública

```rust
pub fn geocentric_solar_longitude(
    earth_heliocentric_longitude: Degrees,
) -> Degrees;
```

La salida queda normalizada al intervalo:

```text
0° <= Lg < 360°
```
