# 1. Objetivo

Calcular el número de **siglos julianos transcurridos desde la época J2000.0**.

Este valor se representa tradicionalmente con la variable:

```text
T
```

T constituye la variable independiente de la mayoría de los algoritmos astronómicos modernos descritos por Jean Meeus.

---

# 2. Definición

Sea:

- **JD** = Julian Day del instante considerado.

La época J2000.0 corresponde exactamente a:

```text
JD = 2451545.0
```

Entonces:

```text
T = (JD - 2451545.0) / 36525.0
```

donde:

```text
36525 = número de días de un siglo juliano
```

---

# 3. Interpretación

Si:

```text
JD = 2451545.0
```

entonces:

```text
T = 0
```

Un siglo después:

```text
JD = 2488070.0
```

entonces:

```text
T = 1
```

Un siglo antes:

```text
JD = 2415020.0
```

entonces:

```text
T = -1
```

---

# 4. Precisión

La operación es puramente algebraica.

No introduce aproximaciones.

Toda la precisión depende exclusivamente de la precisión del Julian Day de entrada.

## Rango de validez

La fórmula es una definición algebraica relativa a J2000.0.

El rango práctico de validez queda determinado por:

- la validez del Julian Day de entrada;
- la precisión numérica de `f64`;
- el rango de validez de los algoritmos posteriores que utilicen `T`.

## Tolerancia aceptada

Para pruebas internas exactas de referencia se usa la tolerancia global de
comparación flotante del proyecto.

La tolerancia científica formal queda pendiente hasta contrastar el algoritmo
contra una fuente externa de referencia.

# 5. Implementación Rust

```rust
use crate::core::JulianDate;

/// Returns Julian centuries since J2000.0.
///
/// Reference:
/// Jean Meeus, Astronomical Algorithms, Chapter 25.
pub fn julian_centuries(jd: JulianDate) -> f64 {
    (jd.value() - 2451545.0) / 36525.0
}
```
