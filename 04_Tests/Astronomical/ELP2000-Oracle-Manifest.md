### Algorithm

For each official `ELP1` record:

1. read the four main-problem multipliers;
2. construct the ELP main-problem argument from `D`, `l'`, `l`, and `F`;
3. apply the ELP2000-82B/85 main-problem amplitude correction documented in
   `02_Research/ELP2000/notas/ELP1-Semantics.md`;
4. compute the signed contribution as corrected amplitude times sine of the
   argument;
5. write the term index, individual contribution and cumulative partial sum.

The generator does not call Pancanga Engine Rust code.
