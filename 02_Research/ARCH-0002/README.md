# Investigación Asociada a ARCH-0002

Esta carpeta reúne referencias y notas para la arquitectura del subsistema
astronómico.

## Estructura

- `papers/` - papers, estándares y documentación técnica.
- `links/` - enlaces a motores y referencias externas.
- `notas/` - notas de revisión y decisiones pendientes.
- `ejemplos/` - ejemplos de arquitectura, APIs o datos de referencia.
