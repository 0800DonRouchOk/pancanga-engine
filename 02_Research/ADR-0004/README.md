# Investigación Asociada a ADR-0004

Esta carpeta reúne las fuentes y notas usadas para decidir la adopción de
`VSOP87D` en la cadena solar de Pancanga Engine.

## Estructura

- `papers/` - documentación técnica, papers y fuentes primarias.
- `links/` - enlaces a IMCCE, catálogos y referencias externas.
- `notas/` - notas de revisión y análisis comparativo.
- `ejemplos/` - posibles casos de validación y datos de referencia.

## Decisión Asociada

`ADR-0004` adopta `VSOP87D` para calcular la posición heliocéntrica de la Tierra
en coordenadas esféricas referidas a la eclíptica y equinoccio de la fecha.
